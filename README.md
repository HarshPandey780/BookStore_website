<h1>Book Store Website</h1>
This is a full-stack web application for a book store built using the MERN stack (MongoDB, Express.js, React.js, Node.js). 
<h2>Technology Used</h2>
<ul>
  <li>React</li>
    <li>Express</li>
    <li>Mongoose</li>
    <li>Javascript</li>
    <li>Node Js</li>
    <li>JWT Token</li>
    
</ul>
<h2> Features</h2>
<ul>
  <li>Login Page</li>
    <li>Add Books</li>
    <li>Add Students</li>
    <li>Dashboardt</li>
    <li>Delete Books</li>
    
    
</ul>
